# אתר לימוד לדוגמה

ברוכים הבאים לאתר עברי קטן שנועד להדגים איך מורה יכולה לנהל חומרי לימוד ב-Markdown ולפרסם אותם דרך GitHub Pages.

{: .note }
האתר נבנה כך שאפשר להוסיף שיעורים חדשים כקבצי Markdown פשוטים, ללא צורך בהפעלת שרת מקומי.

## מה תמצאו כאן

| אזור | מטרה |
| --- | --- |
| [אודות]({{ site.baseurl }}/about/) | הסבר קצר על האתר |
| [שיעור פתיחה]({{ site.baseurl }}/lessons/intro/) | דוגמה למבנה שיעור |
| [שיעור Markdown]({{ site.baseurl }}/lessons/markdown-basics/) | דוגמה לכתיבה וארגון תוכן |

## דוגמת קוד

קוד, שמות קבצים ופקודות נשארים משמאל לימין:

```html
<h1>שלום כיתה</h1>
<p>זהו קטע HTML לדוגמה.</p>
```
