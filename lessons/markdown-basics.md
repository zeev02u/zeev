# יסודות Markdown

Markdown מאפשר לכתוב תוכן ברור בלי להתעסק בעיצוב מורכב.

## תחביר שימושי

| פעולה | דוגמה |
| --- | --- |
| כותרת | `## כותרת משנה` |
| הדגשה | `**טקסט מודגש**` |
| קישור | `[שם הקישור](https://example.com)` |

## דוגמה

```md
## משימה

- קראו את הטקסט.
- כתבו שתי שאלות.
- שתפו תשובה אחת בכיתה.
```

לאחר שמירת הקובץ ודחיפה ל-GitHub, תהליך הפרסום יעדכן את האתר.
