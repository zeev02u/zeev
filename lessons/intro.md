# שיעור פתיחה

מטרת השיעור היא להכיר את מבנה האתר ואת דרך העבודה עם קבצי Markdown.

## מטרות

1. להבין מהו עמוד Markdown.
2. לזהות את אזור ה-front matter.
3. לערוך טקסט, קישורים ורשימות.

{: .tip }
כדאי לשמור כל שיעור בקובץ נפרד תחת התיקייה `lessons/`.

## פעילות קצרה

פתחו קובץ שיעור, שנו את הכותרת בשדה `title`, והוסיפו פסקת פתיחה שמתארת את מטרת השיעור.
