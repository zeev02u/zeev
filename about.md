# אודות

האתר הזה הוא תבנית בסיסית לאתר לימודי בעברית. הוא מיועד לסדנה שבה התוכן נערך בקבצי Markdown, והפרסום מתבצע אוטומטית דרך GitHub Actions ו-GitHub Pages.

## עקרונות עבודה

- כל עמוד כולל front matter בתחילת הקובץ.
- התוכן הראשי בעברית ומוצג מימין לשמאל.
- מונחים טכניים כמו `index.md`, `git push` ו-`nav_order` נשארים משמאל לימין.
- אין להעלות מידע פרטי של תלמידים, הורים או צוות בית הספר.
